# Serwis RTV/AGD

Projekt na kurs Bazy danych

# Tech stack
## Frontend:
- Angular v15
## Backend:
- Java v17
- Spring Boot v3.0.4
## Baza danych:
- Postgresql v15.1 (w przyszłości baza Oracle)
- Hibernate (jako ORM)

# Uruchomienie
## Backend:
```bash
cd ./backend
```
```bash
mvn clean install
```
```bash
mvn spring-boot:run
```
